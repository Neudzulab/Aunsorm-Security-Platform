# Blockchain Test Ajanı

- Bu dizindeki senaryolar, blockchain PoC entegrasyonlarını doğrulayan deterministik test vektörleri içindir.
- Mock ledger implementasyonları, saldırgan senaryolarını da kapsayan bütünlük kontrolleri sağlamalıdır.
- Test verisi ve konfigürasyon örnekleri (örn. `config.example.toml`) gerçek gizli anahtarlar içermemeli, açıklama satırlarıyla belgelendirilmelidir.

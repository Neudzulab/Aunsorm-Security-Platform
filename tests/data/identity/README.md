# Identity Flow Fixtures

Bu dizin, kimlik entegrasyonu testleri için deterministik JSON
fixture'ları içerir. Her fixture, JWT imzalama, KMS yerel store
özellikleri ve X.509 kalibrasyon parametreleri için beklenen
çıktıları tanımlar. Dosyalar CI ortamında ağ erişimine ihtiyaç
duymadan yeniden üretilebilir olacak şekilde tasarlandı.

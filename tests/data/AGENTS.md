# Test Veri Rehberi

Bu dizin yalnızca deterministik test girdileri ve fixture dosyaları
barındırır. Dosyalar JSON veya benzeri okunabilir formatlarda
olmalı, dış servis erişimi gerektirmemelidir. Her fixture, üretim
kaynağı ve amaçlanan test kapsamını açıklayan kısa bir belge ile
beraber tutulmalıdır.

# Certification Agent Yönergeleri

Bu dizin, sertifikasyon PoC ve uyumluluk senaryolarını belgelemekten sorumlu "Certification Agent" tarafından yönetilir.

- Senaryolar NIST, ETSI/ENISA, BSI gibi kurumların yayımladığı çerçevelere referans vermelidir.
- Fixture dosyaları mümkün olduğunca açıklayıcı meta veri alanları içermeli ve ileri analizlere temel oluşturmalıdır.
- Test betikleri mevcut üretim kodunu değiştirmeden PoC düzeyinde doğrulamalar yapmalıdır.
